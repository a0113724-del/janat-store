لوحة تحكم المتجر

الصق مفتاح الدخول الخاص فيك (GitHub Token)

المفتاح غير صحيح أو ما عنده صلاحية — تأكد إنك نسخته كامل
أول مرة؟ سوي مفتاح جديد:
1. روح لـ github.com/settings/tokens?type=beta
2. اضغط "Generate new token"
3. باسم Token: اكتب أي اسم (مثلاً "لوحة المتجر")
4. Repository access: اختر "Only select repositories" → janat-store
5. Permissions → Contents → اختارها "Read and write"
6. اضغط "Generate token" وانسخ المفتاح (يبدأ بـ github_pat_) والصقه فوق
